<h2>Сайт планировщик задач (to-do list)</h2>
Функционал включает в себя:
<ul>
  <li>Создание, редактирование, удаление задач</li>
  <li>Создание списков задач</li>
  <li>Приоритеты и сроки выполнения</li>
  <li>Функционал совместной работы</li>
  <li>Уведомление о дедлайнах</li>
</ul>

<p>Работа велась в 2-х фреймворках: Laravel (back) и Vue.js (front). Также использовались MySQL (БД), nginx, redis. В качестве платформы был использован Docker.</p>
